> 개발 기간: 2024. 1. 3 ~ 2025. 2. 21  

**"Wedit: Wedding + Edit"**

디지털로 만나는 나만의 특별한 커스텀 모바일 청첩장

예비 부부들이 디지털 환경에서 쉽고 빠르게 결혼 소식을 알리고,
자신만의 개성있는 모바일 청첩장을 제작할 수 있도록 돕는 웹 서비스
